Centos7系统用 VSCode 编译和调试 C++ 源码

发表于 2022-10-21 | 阅读量 次
| 字数统计: 353 | 阅读时长 ≈ 2

安装 CMake

  1. 安装gcc等依赖程序
    yum install -y gcc gcc-c++ wget

  2. 获取CMake源码包
    wget https://cmake.org/files/v3.23/cmake-3.23.2.tar.gz

  3. 解压并安装

安装 VSCode

  1. yum 命令安装

  2. 下载 rpm 包安装
    复制链接下载最新 rpm 包：https://update.code.visualstudio.com/1.70.1/linux-rpm-x64/stable
    安装：rpm -ivh code-1.70.1-1660113182.el7.x86_64.rpm

启动 VSCode：

  1. 通过桌面程序启动
  2. 通过终端启动
  3. 点击左侧插件菜单栏搜索C++安装扩展包

将源码通过 CMake 编译

  1. 打开源码文件夹
  2. 点击 cmake 菜单栏，点击config project
  3. config完成后会生成如下目录、再点击如图build生成二进制文件

开始调试

  1. 点击调试菜单、生成launch.json文件

  2. 配置调试参数

  3. 设置断点，启动调试
