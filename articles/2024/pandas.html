判断第二次的值是否比第一次小或不大于10%

此时需要用到 numpy 库的 where 函数，

pip install numpy

numpy.where(condition, x, y)

  • condition ：表示条件的布尔数组或条件表达式。
  • x ：满足条件时的返回值，可以是数组或标量。
  • y ：不满足条件时的返回值，可以是数组或标量。
