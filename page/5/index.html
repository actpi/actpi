nginx-proxy 介绍

nginx-proxy 启动一个容器来运行 nginx 和 docker-gen。 在主机上的容器启动和停止时 docker-gen 会生成 nginx 反向代理配置并且重新加载 nginx。

  • /etc/nginx/certs 创建和更新 Let’s Encrypt 证书

iis服务器 error ；“/”应用程序中的服务器错误。

发表于 2017-10-12 本文字数： 102 阅读时长 ≈ 1 分钟

问题

windows server 上的 IIS 服务器报错：“/”应用程序中的服务器错误。
因为网页是 aspx 格式，网站首页为 default.aspx, 浏览器里调试提示 找不到 default.aspx 。

解决办法

打开 IIS 管理程序，找到所属网站站点，在设置界面找到默认页面，设置首页类型里 把 aspx 格式调至第一位置。
重启即可解决！
